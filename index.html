How to setup?

  1. Add the typescript file to be compiled or the already complied javascript file to your project
  2. Add the class "sortable" to a table you want to make sortable
  3. Customise the class names within the "USER EDITABLE AREA" within the js/ts file
  4. Add your own CSS and you're away!
